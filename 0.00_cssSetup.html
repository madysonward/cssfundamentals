CSS Notes

The Purpose of CSS is...

  • CSS can be included within html elements using the style attribute (or, attr).
  • CSS can be used with the style element in html.
  • CSS is for altering colors, fonts, backgrounds, etc.
  • It can also implement some effects and animations.

CSS Statements

The four main components of CSS include the following:

  1. Declaration: The statement as a whole with curly braces.
  2. Selector: The name of the part you want to style.
  3. Property: The properties you want to edit (ex: background, height, width, etc).
  4. Value: Numbers, colors, urls, - the values we assign.
