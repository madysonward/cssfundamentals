IDs

An ID is a unique identifier.

ID Notes

  • It's unique to the element that it's on.
  • To target it from our CSS, we need to use a # sign. Ex: #customId instead of .customId
  • You can use them with classes on the same elements!
  • IDs are most commonly used with JavaScript, rather than classes.
  • If you have a class, and an ID on the same element, the ID will take priority over the class.
